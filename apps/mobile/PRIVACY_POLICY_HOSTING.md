# 📋 Datenschutzerklärung für App Store Connect - Hosting-Anleitung

## 🎯 Problem gelöst!

Du hast bereits eine vollständige Datenschutzerklärung in `privacy-policy.html`. Jetzt musst du sie nur noch online hosten.

## 🚀 Einfachste Lösung: GitHub Pages (Kostenlos)

### **Schritt 1: GitHub Repository erstellen**
1. Gehe zu [github.com](https://github.com)
2. Erstelle ein neues Repository: `nicht-rauchen-privacy`
3. Lade die `privacy-policy.html` hoch

### **Schritt 2: GitHub Pages aktivieren**
1. Gehe zu Repository → Settings
2. Scrolle zu "Pages"
3. Source: "Deploy from a branch"
4. Branch: "main"
5. Save

### **Schritt 3: URL erhalten**
Deine Datenschutzerklärung ist dann verfügbar unter:
```
https://deinusername.github.io/nicht-rauchen-privacy/privacy-policy.html
```

## 🌐 Alternative Hosting-Optionen

### **Option 1: Netlify (Kostenlos)**
1. Gehe zu [netlify.com](https://netlify.com)
2. Drag & Drop die `privacy-policy.html`
3. Erhalte sofort eine URL

### **Option 2: Vercel (Kostenlos)**
1. Gehe zu [vercel.com](https://vercel.com)
2. Import GitHub Repository
3. Deploy automatisch

### **Option 3: Firebase Hosting (Kostenlos)**
1. Gehe zu [firebase.google.com](https://firebase.google.com)
2. Erstelle neues Projekt
3. Aktiviere Hosting
4. Deploy die HTML-Datei

## 🔗 App Store Connect konfigurieren

### **Schritt 1: App Store Connect öffnen**
1. Gehe zu [appstoreconnect.apple.com](https://appstoreconnect.apple.com)
2. Wähle deine App "Nicht Rauchen"

### **Schritt 2: Privacy Policy URL eingeben**
1. Gehe zu "App Information"
2. Scrolle zu "Privacy Policy URL"
3. Füge deine URL ein:
   ```
   https://deinusername.github.io/nicht-rauchen-privacy/privacy-policy.html
   ```

### **Schritt 3: Speichern**
1. Klicke "Save"
2. Die URL ist jetzt für Apple verfügbar

- ✅ DSGVO-konform

### **App Store Requirements:**
- ✅ Privacy Policy URL erforderlich
- ✅ Muss öffentlich zugänglich sein
