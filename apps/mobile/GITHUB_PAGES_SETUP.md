# 🚀 GitHub Page für Datenschutzerklärung erstellen

## 📋 Schnellste Lösung (5 Minuten)

### **Schritt 1: GitHub Repository erstellen**
1. Gehe zu [github.com](https://github.com) und logge dich ein
2. Klicke auf das **"+"** Symbol oben rechts → **"New repository"**
3. Repository Name: `nicht-rauchen-privacy`
4. Beschreibung: `Datenschutzerklärung für Nicht Rauchen App`
5. Wähle **"Public"** (wichtig für GitHub Pages)
6. **NICHT** "Add a README file" anhaken
7. Klicke **"Create repository"**

### **Schritt 2: HTML-Datei hochladen**
1. Klicke **"uploading an existing file"**
2. Ziehe die `privacy-policy.html` in den Upload-Bereich
3. Commit message: `Add privacy policy`
4. Klicke **"Commit changes"**

### **Schritt 3: GitHub Pages aktivieren**
1. Gehe zu **Settings** (Tab oben)
2. Scrolle runter zu **"Pages"** (linke Sidebar)
3. Bei **"Source"**: Wähle **"Deploy from a branch"**
4. Bei **"Branch"**: Wähle **"main"**
5. Bei **"Folder"**: Wähle **"/ (root)"**
6. Klicke **"Save"**

### **Schritt 4: URL erhalten**
Nach 1-2 Minuten ist deine Seite verfügbar unter:
```
https://deinusername.github.io/nicht-rauchen-privacy/privacy-policy.html
```

## 🎯 App Store Connect konfigurieren

### **Schritt 1: App Store Connect öffnen**
1. Gehe zu [appstoreconnect.apple.com](https://appstoreconnect.apple.com)
2. Wähle deine App "Nicht Rauchen"

### **Schritt 2: Privacy Policy URL eingeben**
1. Gehe zu "App Information"
2. Scrolle zu "Privacy Policy URL"
3. Füge deine URL ein:
   ```
   https://deinusername.github.io/nicht-rauchen-privacy/privacy-policy.html
   ```

### **Schritt 3: Speichern**
1. Klicke "Save"
2. Die URL ist jetzt für Apple verfügbar


## ⚠️ Wichtige Hinweise

- **Repository muss Public sein** für GitHub Pages
- **Warte 1-2 Minuten** nach dem Aktivieren von Pages
- **Teste die URL** bevor du sie in App Store Connect einfügst
- **Speichere die URL** für spätere Verwendung
