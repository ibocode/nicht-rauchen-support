# SUPPORT-URL FÜR APP STORE

## 🎯 WAS DU BRAUCHST:

Du brauchst eine **Support-URL** für deine App. Das ist eine Webseite mit Support-Informationen, die im App Store angezeigt wird.

## 📋 OPTIONEN FÜR SUPPORT-URL:

### Option 1: GitHub Pages (KOSTENLOS)
- **Erstelle eine GitHub Repository** für deine App
- **Aktiviere GitHub Pages**
- **Erstelle eine Support-Seite** mit FAQs und Kontakt

### Option 2: Netlify/Vercel (KOSTENLOS)
- **Erstelle eine einfache HTML-Seite**
- **Hoste sie auf Netlify oder Vercel**
- **Füge Support-Informationen hinzu**

### Option 3: Eigene Domain (KOSTENPFlichtig)
- **Kaufe eine Domain** (z.B. nichtrauchen-app.de)
- **Erstelle eine Support-Seite**
- **Hoste sie auf einem Server**

## 🚀 SCHNELLSTE LÖSUNG - GITHUB PAGES:

### 1. GitHub Repository erstellen:
- **Name:** nicht-rauchen-app-support
- **Öffentlich** machen
- **README.md** mit Support-Informationen

### 2. GitHub Pages aktivieren:
- **Settings** → **Pages**
- **Source:** Deploy from a branch
- **Branch:** main
- **URL:** https://deinusername.github.io/nicht-rauchen-app-support

### 3. Support-Seite erstellen:
```html
<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Nicht Rauchen - Support</title>
</head>
<body>
    <h1>Nicht Rauchen - Support</h1>
    
    <h2>Häufige Fragen (FAQ)</h2>
    <h3>Wie funktioniert die App?</h3>
    <p>Die App hilft dir beim Rauchstopp durch tägliche Check-ins und Fortschrittsverfolgung.</p>
    
    <h3>Wie speichere ich meine Daten?</h3>
    <p>Alle Daten werden lokal auf deinem Gerät gespeichert. Keine externen Services erforderlich.</p>
    
    <h3>Wie aktiviere ich Benachrichtigungen?</h3>
    <p>Gehe zu Einstellungen und aktiviere "Tägliche Erinnerungen".</p>
    
    <h2>Kontakt</h2>
    <p>Bei Fragen oder Problemen:</p>
    <p>E-Mail: support@nichtrauchen-app.de</p>
    
    <h2>Datenschutz</h2>
    <p>Alle Daten werden lokal gespeichert. Keine Datenübertragung an externe Server.</p>
</body>
</html>
```

## ⚠️ WICHTIG:

- **URL muss erreichbar** sein
- **HTTPS verwenden** (https://)
- **Support-Informationen** enthalten
- **Kontaktmöglichkeit** anbieten
